# Tab Out 2.0

A Chrome extension that replaces the new tab page with a clean dashboard of your open tabs, grouped by domain.

This is a fork of [Tab Out](https://github.com/zarazhangrui/tab-out) by [Zara](https://x.com/zarazhangrui), extended with the following features:

## New in 2.0

- **Dark / Light / System theme** — toggle in the footer, persists across sessions
- **Customizable Homepages sites** — add or remove sites from the Homepages group via the settings panel (no code required)
- **Auto-refresh** — dashboard updates automatically when tabs open or close
- **Tab search** — real-time filter in the header; press `/` to focus
- **Keyboard shortcuts** — `/` to focus search, `Escape` to clear or close the settings panel
- **Archive auto-cleanup** — completed items older than 30 days are pruned automatically; manual "Clear archive" button in settings

## Install

1. Clone or download this repo
2. Open `chrome://extensions` and enable **Developer mode**
3. Click **Load unpacked** and select the `extension/` folder
